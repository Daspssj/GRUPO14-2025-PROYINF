
Ejecucion programa: Para ejecutar correctamente el sitio, se debe descomprimir el archivo principal, y posteriormente agregarlo a la carpeta "htdocs" de xampp
Luego, se deben extraer los php de la carpeta "php", luego se debe importar la base de datos a una llamada
"ensayos_paes" para que de esta forma se conecte a la pagina web correctamente. Ya con esto, debe iniciar Xampp Control Panel y activar los puertos mysql y apache,
ya con esto solo se debe conectar al archivo mediante el enlace "localhost/EnsayoPaes/login.php" y listo.
Ahora disfrutar de ensayar o crear ensayos.
